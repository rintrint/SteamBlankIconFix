# SteamBlankIconFix

修復Steam遊戲圖示變成空白的問題。

## 步驟

1. 運行 script/1_fix_registry.reg
2. 運行 script/2_clear_thumbnail_cache.bat

運行完兩個腳本後觀察問題是否已經解決。
